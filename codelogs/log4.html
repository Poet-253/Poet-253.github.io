HTML Semantic Tags

What do these CSS properties do?

    color
    background-color
    border-color
The different color tags all change the color of specific areas in the html.
Non-semantic tags are tags in HTML that aren't specific to the content they'll include. While they can be used to achieve the same goal as using semantic tags, non-semantic tags can often make the code look overly complicated and difficult to navigate.

Why Choose Semantic Tags?

    Benefits to using semantic tags are convenience based.
    They give direction, style, models, grouping, and design in easy-to-understand format to allow for easy adjustment and pinpointing problems to be fixed.
    Instead of sifting through div and span tags, you would only need to locate the specified paragraph, heading, etc.
